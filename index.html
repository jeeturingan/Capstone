Welcome!


The 3D Braille System is created empowered by Java. With the use of the system, users can create 3D Tactile Storyboard using the object templates and users can also add braille text by letting the system convert text to braille text. Users can also customize their own storyboards by adjusting different settings on the Manual tab. The system also has a converter that allows the conversion of 2D images to 3D object models. The user can upload an image and the system can convert it automatically.

The 3D Tactile Storyboard is made intentionally for those who are visually impaired. The system aims to create more learning materials for the visually impaired and as well as the teachers that will guide and teach them.
